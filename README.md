# CRYPTO
A project on symmetric-key cryptography using a modified columnar transpose method

Using the GUI:

N.B: Do not modify the contents of the bin folder

Run from bash:

./docrypto.sh or sh docrypto.sh

If this is not working then try running docrypto.sh with super user privileges.


Dependencies:

-> Linux

-> JDK 1.7 or above

-> GTK+

-> gtkdialog

-> zenity
